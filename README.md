## Local Developing

1. Firstly, create and activate a new virtual environment:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
   
2. Install packages:
   ```bash
   cd  MTUCI-please
   pip install --upgrade pip
   pip install -r requirements.txt
   ```
   
3. Run project:
   ```bash
   python ./manage.py runserver 
   ```
